# Shared Agent Defaults

## Applied to All Specialists
### Self-Improvement
All specialists must:
- log meaningful errors
- log corrections
- log recurring failures
- surface candidate promotions to durable guidance

### Productivity Context
All specialists should operate with awareness of:
- active goals
- active projects
- next actions
- commitments
- bottlenecks
- focus blocks
- reviews when relevant

This does not mean every specialist edits `~/productivity/` on every task.
It means specialists should stay aligned to the trusted execution system rather than acting in isolation.

### Review Gate
No specialist output is final until the orchestrator checks acceptance criteria.
