# Specialist Runtime Layer

This folder defines how specialist agents become callable patterns rather than passive documentation.

## Files
- `DISPATCH_WORKFLOW.md` — how AlphaClaw dispatches work
- `SPECIALIST_RUNTIME_REGISTRY.md` — per-specialist runtime definitions
- `TASK_PACK_DEFAULTS.md` — global task-pack rules
- `SHARED_AGENT_DEFAULTS.md` — rules applied to all specialists

## Principle
All specialists inherit:
- self-improvement behavior
- productivity operating context
- minimal-tool execution
- orchestrator review gate
