# Task Pack Defaults

## Every specialist task pack should include
- TASK
- GOAL
- CONTEXT
- ACCEPTANCE CRITERIA
- OUT OF SCOPE
- RISK FLAGS
- TOOLS ALLOWED
- MEMORY INPUTS
- REPORT FORMAT

## Shared Defaults
- Keep scope narrow.
- Prefer smallest useful step.
- Route unrelated work back to orchestrator.
- If blocked, report exact blocker and exact need.
