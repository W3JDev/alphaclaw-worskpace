# Dispatch Checklist

Before dispatching any specialist:
- [ ] Verified skill state in `SKILLS_INVENTORY.md`
- [ ] Verified specialist exists in `AGENT_REGISTRY.md` or route is explicitly orchestrator-managed
- [ ] Selected runtime (ACP vs Pi vs local tool path)
- [ ] Prepared bounded task pack
- [ ] Limited tools to task minimum
- [ ] Defined acceptance criteria
- [ ] Determined if self-improvement writeback may be needed
- [ ] Determined if productivity system should be updated
