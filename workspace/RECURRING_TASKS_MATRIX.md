# RECURRING_TASKS_MATRIX

## Daily
- Inbox triage
- Calendar/meeting prep
- Priority extraction
- Lead / pipeline review
- Customer escalation review
- Content draft queue
- Ops health check

## Weekly
- Team summary
- Overdue tasks review
- Follow-up batch
- KPI refresh
- Content planning
- Pipeline review
- Backlog grooming

## Biweekly
- Sprint / ops review
- Bottleneck scan
- Lead quality review
- Support pattern review

## Monthly
- Executive report
- Agent quality audit
- Automation ROI review
- Memory distillation
- Template improvement review
- Client ops playbook update

## Rule
Every recurring task must map to:
- owner agent
- trigger/source
- output file/system
- approval level
- escalation path
